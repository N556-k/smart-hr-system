# Smart HR System

A simple and user-friendly Human Resource Management System (HRMS) built with **Flask** and **SQLite**.

## 🚀 Features

- Add employees (name, department, salary)
- Delete employees
- Display employee list
- Responsive Bootstrap design
- Easy to run and modify

## 🛠️ Technologies Used

- Python 3
- Flask
- SQLAlchemy
- SQLite
- Bootstrap 5
